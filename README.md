🏆 Copadex - Meu Álbum 2026

Copadex (Meu Álbum 2026) é um aplicativo Android 100% offline desenvolvido para auxiliar os colecionadores a gerenciarem suas figurinhas da Copa do Mundo FIFA de 2026 de forma simples, rápida e privada.

📱 Capturas de Tela

🎯 Sobre o Projeto

Este projeto foi desenvolvido como parte de um trabalho acadêmico (disciplina de SPD - Consultoria Empresarial). O objetivo principal era criar uma solução mobile focada em usabilidade e performance, que resolvesse um problema real: o gerenciamento de figurinhas em locais de troca físicos, onde a conexão com a internet costuma ser instável ou inexistente.

Por isso, o Copadex nasceu sob a premissa de ser um aplicativo Standalone (100% Offline), garantindo que os dados do usuário fiquem salvos apenas na memória do seu próprio dispositivo, oferecendo privacidade total e zero dependência de servidores externos.

✨ Principais Funcionalidades

📊 Dashboard Dinâmico: Visão geral do álbum com gráfico circular de progresso, total de figurinhas obtidas, faltantes e repetidas.

📚 Controle de Grade: Adicione ou remova figurinhas do seu álbum com um sistema inteligente de cliques (Faltante -> Obtida -> Repetida).

🔄 Gestão de Repetidas: Uma aba exclusiva para gerenciar o saldo de figurinhas disponíveis para troca, facilitando negociações presenciais.

👥 Múltiplos Perfis: Crie e gerencie mais de um álbum no mesmo dispositivo (ex: o seu álbum e o álbum de um familiar), sem misturar os dados.

🛡️ Privacidade Local: Sem necessidade de login, nuvem ou internet.

🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as ferramentas mais modernas do ecossistema Android:

Kotlin: Linguagem de programação moderna, segura e concisa.

Jetpack Compose: Kit de ferramentas nativo do Android para a construção de interfaces (UI) de forma declarativa.

Material Design 3: Padrões de design para garantir uma interface bonita, acessível e responsiva.

Arquitetura Baseada em Componentes: Divisão lógica de estados usando a reatividade do Compose (remember, State).

🚀 Como Executar o Projeto

Para testar o projeto na sua própria máquina, siga os passos abaixo:

Pré-requisitos

Android Studio (Versão mais recente recomendada, ex: Hedgehog ou Iguana).

SDK do Android API 34+ configurado.
